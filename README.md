# Learn English

**Learn English** is a simple and beginner-friendly web application designed to help users improve their English vocabulary in an easy and interactive way. The platform provides dynamic vocabulary cards and a clean user interface to make learning more engaging.

## 🌟 Features
- 📚 Vocabulary learning with dynamic cards
- 📝 Practice exercises for better understanding
- 🎯 Beginner-friendly and clean user interface
- ⚡ Fast and responsive design

## 💻 Technology Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📖 About This Project
This project was developed to practice frontend development and build a simple learning platform for beginners who want to improve their English vocabulary. It focuses on creating a clean interface and an easy learning experience.
